def soma(a,b) = cria função que soma dois valores, return = devolve o resultado
def mult(a,b) = cria função que multiplica valores
print(f"...") = imprime mensagem formatada com variável
if a > b = compara valores e mostra o maior
// = divisão inteira, % = resto da divisão, return q,r = retorna dois valores
a % 2 == 0 = verifica se número é par (retorna True/False)
função sem return = retorna None automaticamente
parâmetros posicionais = seguem a ordem definida na função
parâmetros nomeados (nome=...) = não dependem da ordem
p="dia" = valor padrão (default)
parâmetro vazio pode ser tratado com if
parâmetro com valor padrão não pode vir antes de obrigatório
*numeros = aceita vários valores como tupla, sum() soma tudo
**dados = recebe argumentos nomeados como dicionário, .items() percorre
*args = vários argumentos posicionais, **kwargs = vários argumentos nomeados
variável dentro da função ≠ variável fora 
recursão = função chama ela mesma até condição de parada
variável global pode ser acessada dentro da função
função pode receber outra função como parâmetro 
lambda = função anônima rápida (x: x**2)
map() = aplica função a todos elementos da lista
filter() = filtra elementos com base em condição
recursão com multiplicação = fatorial (n * f(n-1))
recursão de contagem = imprime até chegar no fim
erro de recursão infinita = falta condição de parada correta
""" """ = docstring (documentação da função)
help() = mostra documentação da função
if/elif = escolhe operação (calculadora simples)
*args e **kwargs juntos = recebe qualquer tipo de argumento
